# 动画效果

使用anime.js来做。

## 展示方法：

> 注：建议使用`yarn`进行包管理。

1. 安装parcel。使用`yarn`：`yarn global add parcel-bundler`；使用`npm`：`npm i -g parcel-bundler`；
2. 然后`parcel build ./index.html`即可展示。
